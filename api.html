class blocknative.stream.Stream(api_key: str, blockchain: str = 'ethereum', network_id: int = 1, global_filters: Optional[List[dict]] = None)

Stream class used to connect to Blocknative’s WebSocket API.

Parameters

  • api_key – The api key. Get one at blocknative.com.

  • blockchain – The blockchain you want to connect to. Default is ethereum.

  • network_id – The id of the network. For instance, 4 for Ethereum Rinkeby.

  • global_filters – The filters that will be applied globally to the stream.

blocknative.stream.Stream.subscribe_address(self, address: str, callback: Callable[[dict, Callable], None], filters: Optional[List[dict]] = None, abi: Optional[Union[List[dict], str]] = None)

Subscribes to an address to listen to any incoming and outgoing transactions that occur on that address.

Parameters

  • address – The address to watch for incoming and outgoing transactions.

  • callback – The callback function that will get executed for this subscription.

  • filters – The filters by which to filter the transactions associated with the address.

  • abi – The ABI of the contract. Used if address is a contract address.

stream = Stream(API_KEY)

# Define the address you want to watch
address = '0x7a250d5630b4cf539739df2c5dacb4c659f2488d'

# Define your transaction handler which has the context of a specific subscription.
async def callback(txn, unsubscribe):
    # Output the transaction data to the console
    print(json.dumps(txn, indent=4))

stream.subscribe_address(address, callback, filters=[{"status": "pending"}])

Parameters

  • txn_hash – The hash of the transaction to watch.

  • status – The status of the transaction to receive events for. Leave out for all events.
